기술은 그냥 아이디어를 실현하는 도구일 뿐이다. 자잘한 기술들은 너무 깊게 배우려 하지 마라.

그치만 중심이 되는 기술들은 깊게 배워두는 것이 매우 중요하다. JavaScript 같은 것들이다.

나는 TypeScript는 중심이 된다고는 생각하지 않는다. 그냥 타입을 표현하는 도구이기 때문이다.

나는 Styled Components던 React Router Dom 이던 그냥 그때그때 마다 API 보면서 앱을 만든다. 내가 주로 사용하는 Supabase도 같다.

이런 자잘한 기술들은 배우면 배울 수록 어차피 매월 개선되는 버전이 나오기에 의미가 없다.

그러니 프론트엔드 개발자가 되고 싶다면 중심이 되는 HTML, CSS, JS 기술만 잘 익히면 된다. 그치만 필요에 따라 TS도 익히면 좋긴 할 것이다.
